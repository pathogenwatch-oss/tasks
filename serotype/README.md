# Serotyping

Generic interface to support multiple implementations, normally species-specific.

## Output format

```
{
  "value": "",
  "source": ""
}
```

## Implementations

- sistr (Salmonella)
  source: SISTR

- seroba (Streptococcus pneumoniae)
  source: SEROBA

## Versioning Scheme

$TOOL_NAME-v$TOOL_VERSION

Examples:

- sistr-v1.0.2
- seroba-v1.0.1
